# Contributing to Plantarium

Thank you for your interest in contributing to Plantarium!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/plantarium.git`
3. Create a feature branch: `git checkout -b feature/your-feature`

## Development Setup

Follow the instructions in [SETUP.md](SETUP.md) to set up your development environment.

## Code Style

- Follow Rust's standard formatting: `cargo fmt`
- Run linter: `cargo clippy`
- Write tests for new features

## Commit Messages

Use clear, descriptive commit messages:
- `feat: add plant search functionality`
- `fix: resolve database connection timeout`
- `docs: update API documentation`

## Pull Request Process

1. Ensure all tests pass: `cargo test`
2. Update documentation if needed
3. Submit a pull request to the `main` branch
4. Describe your changes in detail

## Testing

```bash
# Run all tests
cargo test

# Run with coverage
cargo test --coverage
```

## Questions?

- Open an issue for bugs or feature requests
- Join discussions in the repository
